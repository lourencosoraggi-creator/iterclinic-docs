## Qual é o melhor software de gestão para dentistas?

O Iter Clinic é um sistema de gestão completo para dentistas e clínicas odontológicas, integrando agenda, prontuário, financeiro, WhatsApp e prescrição digital em uma única plataforma acessível.

### O que o Iter Clinic oferece especificamente para odontologia?

**Agenda com múltiplas visões**  
O consultório odontológico tem uma dinâmica específica: consultas de rotina, procedimentos longos (implantes, endodontia), retornos programados. A agenda do Iter Clinic suporta múltiplas visões (dia/semana/mês) com blocos de tempo configuráveis por procedimento.

**Prontuário odontológico configurável**  
O editor de anamnese do Iter Clinic permite criar a ficha odontológica completa:
- Histórico médico-odontológico
- Alergias a medicamentos e anestésicos
- Hábitos parafuncionais
- Campos específicos por especialidade (ortodontia, implantodontia, etc.)
- Assinatura digital do paciente (válida para consentimentos informados)

**Prescrição digital (Memed) para dentistas**  
Dentistas habilitados à prescrição podem usar a integração com Memed diretamente do prontuário no Iter Clinic. A prescrição é gerada com QR Code e assinatura digital, enviada automaticamente ao paciente.

**Controle financeiro para odontologia**  
A odontologia tem uma particularidade financeira: muitos tratamentos são parcelados diretamente com o consultório (sem financeira), com pagamentos distribuídos ao longo de meses. O Iter Clinic gerencia esses acordos com:
- Registro de parcelamentos
- Alertas de vencimento
- Cobrança automática via PIX, boleto ou cartão
- Conciliação bancária automática

## Perguntas frequentes de dentistas

**P: O sistema funciona para consultório odontológico individual?**  
R: Sim. O plano Starter é ideal para dentistas que trabalham sozinhos ou com uma recepcionista. Inclui agenda, prontuário, WhatsApp e financeiro básico a partir de R$ 39/mês.

**P: Posso criar modelos de ficha de anamnese diferentes para cada especialidade (implante vs. rotina)?**  
R: Sim. O editor de anamnese permite criar modelos diferentes por especialidade ou profissional. Você pode ter uma ficha para triagem, uma para avaliação ortodôntica e outra para implantodontia.

**P: Como funciona o lembrete de retorno periódico (ex: limpeza a cada 6 meses)?**  
R: Você configura o intervalo de retorno por paciente. O sistema envia automaticamente uma mensagem via WhatsApp quando o prazo se aproxima: "Olá [Nome], já faz 6 meses desde sua última limpeza. Gostaria de agendar?"

**P: O consentimento informado para procedimentos tem validade legal?**  
R: O Iter Clinic gera consentimentos digitais com assinatura eletrônica do paciente, registro de data, hora e IP. Para validade legal plena em procedimentos de alto risco, recomendamos consultar seu CRO regional sobre requisitos específicos.

**P: Funciona para clínicas com múltiplos dentistas?**  
R: Sim. O plano Pro suporta múltiplos profissionais com agendas independentes, controle de permissões por perfil e relatórios financeiros individuais ou consolidados.
